3. ANTI-VVIRUS APP: This project uses the combined concept of signature
byte code detection and pattern matching to detect
the presence of any malware in the android device
and helps getting rid of it so that the user
can make the optimum use of the device. It scans
various files and checks them against a viral code
to detect any signs of a virus or malware. If a
virus is found the software app performs one of the two
actions which are: Quarantining- Separating the affected
files into a separate quarantine zone. and
Deleting- Simply deleting the affected files.(2020)
